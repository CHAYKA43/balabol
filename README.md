# balabol
Генератор текста на основе входящих и обученных данных.
## Как он работает?
Для начала он генерирует таблицу рейтинга символов из ```raitingchars```, дальше ему нужен "толчок" и он на основе рейтинга вашего сообщения ищет медиану и использует её в качестве "опоры".
Дальше он проходится по всем словам в памяти и присваивает слову рейтинг основываясь на таблице рейтинга, если рейтинг слова превышает ```minimal_r``` то оно добавляется в ответ.
С шансом 10% балабол составит новую комбинацию из слов и обучится на ней, с шансом ~5% он поменяет слова в своем ответе.
